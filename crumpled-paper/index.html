Crumpled Paper Simulation

Link to Github Repository

About

I love the look of crumpled paper wanted to create a simulator that would create 3D paper models to be used for Paper2D games.

Architecture

Originally, I planned to use research papers to simulate the physics, but I felt that may be overcomplicating the system. I want the "feel" of the paper, not necessarily the physics. I spent a good amount of time just crumpling paper to figure this out. The models are created using Godot. I use Godot's shading language (similar to GLSL) to create a spatial shader modify mesh cubes or planes to create the crumpled look.

System

The system primiarly relies on a modified version of Voronoi noise to create the crumpled look of the paper. Along with the intial cells created from Voronoi noise, I scale the x and z coordinates with a "scrunch" factor. I also added a system in to "pinch" the paper at a certain location to create a crease. The normals of the paper are calcuated just using the height, but can be modified to increase the darkness of the creases. The most up to date version of the shader is "paperv2.gdshader" in the Git repo.

Parameters include

  • Voroni columns/rows
  • Voroni smoothness
  • paper height
  • x&z "scrunch"
  • pinch location
  • pinch strength
  • crease intensity
  • crease brightness

Future Work
